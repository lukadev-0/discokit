# Quick Start

Discokit is an SDK for the [Discord API](https://discord.com/developers/docs/intro). It's a collection of libraries
that make it easier to interact with Discord's API.

## Before we begin

The documentation assumes you have basic knowledge about Node.js and JavaScript.

To create a project manually, we must first install the `discokit` package from npm.

  <Tab>


### Adding a `start` script

Add a `start` script to your `package.json`:

```json {3}
{
  "scripts": {
    "start": "node ."
  }
}
```

### Running the code

To run the code, simply run the following command:

<Tabs items={["npm", "pnpm", "yarn"]}>
  <Tab>

```bash
npm start
```

  </Tab>
  <Tab>

```bash
pnpm start
```

  </Tab>
  <Tab>

```bash
yarn start
```

  </Tab>
</Tabs>

Everytime you make changes, you must stop and restart the bot.
In order to stop the bot, press `CTRL` + `C`, this will terminate the
currently running command.
You can then press the up arrow key to run the previous command.
